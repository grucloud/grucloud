Cluster

Manages an ECS Cluster.

Sample code​

exports.createResources = () => [
  {
    type: "Cluster",
    group: "ECS",
    name: "cluster",
    properties: () => ({
      settings: [
        {
          name: "containerInsights",
          value: "disabled",
        },
      ],
    }),
    dependencies: () => ({
      capacityProviders: ["cp"],
    }),
  },
];

Properties​

  • CreateClusterCommandInput

Dependencies​

  • Capacity Provider
  • KMS Key
