KeyPair

Provide a reference to an SSH key pair, used to connect to EC2 instances.

See the AWS documentation for ec2 key pair to create a new one.

exports.createResources = () => [
  { type: "KeyPair", group: "EC2", name: "kp-ecs" },
];
